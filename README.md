# APS 2026 -- Sistema de Atendimento TI

Este projeto é um sistema de gerenciamento de chamados de suporte técnico desenvolvido em Java, com foco em organização e controle de atendimentos.

## 📌 Objetivo

O sistema tem como objetivo facilitar o cadastro de usuários e técnicos, além da gestão de chamados por categorias, permitindo abertura, alteração de status e listagem de informações.

## ⚙️ Funcionalidades

- Cadastro de usuários e técnicos  
- Cadastro de categorias  
- Abertura de chamados  
- Alteração de status dos chamados  
- Listagem de dados do sistema  

## 🧠 Conceitos aplicados

O projeto foi desenvolvido utilizando:

- Classes e objetos  
- Encapsulamento  
- Construtores  
- Getters e setters  
- Vetores (arrays)  
- Estrutura de menu com switch-case  
- Estruturas de decisão (if) e repetição (while)
